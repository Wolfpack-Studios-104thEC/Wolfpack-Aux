wpec_equipment

==============

##Subaddons

- Equipment PBO is split into subaddons to better organize the mod. 

- **Helmets**
    - Stores all custom helmets

- **Uniforms**
    - Stores all Uniforms and Unit information

- **Backpacks**
    - Stores all backpack and storage information

- **Vests**
    - Stores all vests information

- **NVGs**
    - Stores all NVGs and facewear 

##Macros

- Within the equipment PBO, there is a script_macros.hpp which stores all macro information for this PBO. If you need to expand upon or search how something works, please look there for all information

- Within the equipment PBO, there is a script_component.hpp, this not only stores the information for the PBO to function, but also stores information like armor values, storage values, and other base macros used along all subbadonns.
