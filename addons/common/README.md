# Common

## General
- The common PBO, is well, for common stuff. This pbo houses things that will either be used across multiple PBOs, or is so small it really doesn't deserve its own PBO. For the most part this PBO will also be untouched but still has some fundemental things within it.

### Categories
- The big thing this PBO houses is the `FactionClasses`, `EditorCategories`, and `EditorSubcategories`. 
- All of these are used any time something needs to put in zeus for usage
- For it to work use the provided examples and your good to. If you do not see the proper name, check your macro's for proper use, and check your spelling!
